About This Project

This project is designed to detect colors on urine strips using image processing techniques. Here are the details:

Tools Used

  • Django: A high-level Python web framework that encourages rapid development and clean, pragmatic design.
  • REST framework: A powerful and flexible toolkit for building Web APIs.
  • OpenCV: An open-source computer vision and machine learning software library.
  • NumPy: A fundamental package for scientific computing with Python.
  • HTML/CSS/JavaScript: For creating the frontend of the application.

Algorithm

The algorithm used for detecting colors in the urine strip image involves the following steps:

  1. Upload the image using the frontend form.
  2. Save the uploaded image to the server.
  3. Read the image using OpenCV.
  4. Resize and convert the image to RGB.
  5. Extract the dominant colors using k-means clustering.
  6. Return the RGB values of the detected colors.

Frontend Framework

The frontend is created using vanilla JavaScript, HTML, and CSS. It includes interactive elements such as a file upload form and a results display area.
